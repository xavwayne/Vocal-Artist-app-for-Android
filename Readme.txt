Vocal_Artist_app\app\src\main\java\com\xiaoyuw\vocal_artist_app\model package contains:
   Audience.java: Audience definition class
Vocal_Artist_app\app\src\main\java\com\xiaoyuw\vocal_artist_app\ui package contains:
   FragmentHome.java: A fragment showing nothing
   FragmentInfo.java: A fragment showing the basic information of the artist
   FragmentSong.java: A fragment showing available songs to show
   FragmentVideo.java: A fragment showing available videos to choose
   FragmentWall.java: A fragment showing wall papers to set
   MailList.java: A activity to get on mailing list, register the input data to database
   MainActivity.java: Main activity to show as the first page
   VideoActivity.java: A avtivity to play the selected video
Vocal_Artist_app\app\src\main\java\com\xiaoyuw\vocal_artist_app\util package contains:
   DbHelper.java: Helper class for operation of database
